PROJECT TITLE: Simple Weather Application

DESCRIPTION:
This project is a web-based weather application that allows users to search for weather details of any city.

FEATURES:

* Search weather by city name
* Displays temperature, humidity, weather condition, and wind speed
* Responsive and modern UI

TECHNOLOGIES USED:

* HTML
* CSS
* JavaScript
* Weather API (OpenWeather / wttr.in)

HOW TO RUN:

1. Open index.html in a browser
2. Enter a city name
3. Click search to view weather details
